## 🔥 晚间热门项目

### CodebuffAI/codebuff — 终端里直接用嘴写代码 ⭐ 3,850（今日+275）
- **是什么：** 在终端里说话/输入描述，直接生成代码，不用开 IDE
- **白话原理：**
  - 你在 Terminal 里打一段话描述想要的功能（比如"写个函数把 CSV 转成 JSON"）
  - 它调用大模型（OpenAI / Claude / Gemini）直接生成可用的代码片段
  - 支持主流语言，生成的代码可以直接复制进项目
  - 最核心的创新是"Terminal First"——整个交互完全围绕命令行设计，不依赖任何 GUI
  - 内置语法检查和解释功能，对每个生成的代码块都能说明"为什么这样写"
- **技术栈：** TypeScript + Node.js，插件式模型支持
- **开源协议：** MIT
- **适用人群：** 喜欢在终端干活、不想切出去开浏览器的开发者

### inclusionAI/AReaL — 让大模型自己学会"推理" ⭐ 4,068（今日+173）
- **是什么：** 一个简化版强化学习框架，专门让大语言模型（LLM）学会做复杂推理任务
- **白话原理：**
  - 传统微调让大模型"背答案"，AReaL 让模型自己"想明白"
  - 用 GRPO（一种强化学习算法）训练模型做数学题、代码生成、多步推理
  - 训练过程完全本地化，不需要调用云端 API，自己在自己机器上训
  - 支持主流开源模型（Qwen、Llama、Mistral）直接加载训练
  - 比 OpenAI o1/o3 的在线推理便宜得多——一次训练，永久本地部署
- **技术栈：** Python + PyTorch + HuggingFace Transformers
- **开源协议：** Apache 2.0
- **亮点：** 个人开发者也能训练自己的"推理专用模型"，不用依赖昂贵的付费 API
